# lineage2-adena-hunt
Java Spring Boot web application for managing Adena Hunt events, players and game data.

## About
Lineage2 Adena Hunt is a Java Spring Boot web application inspired by the Lineage II MMORPG.

The application helps players organize Adena farming sessions by managing characters, parties and monster information through a modern Spring MVC architecture.

This project was developed during the **Spring Fundamentals** course at **Software University (SoftUni)**.

## Features
- Player Management
- Party Management
- Monster (Mob) Management
- Character Classes
- Session Management
- CRUD Operations
- Input Validation
- Exception Handling
- MVC Architecture

## Technologies
- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- Hibernate
- MySQL
- Thymeleaf
- Maven

## Architecture

Controller -> Service -> Repository -> Database
